# 1. 专业术语

| 词汇         | 说明                                                                                                                                                                                 |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 情景         | 在车辆驾驶中, 用于描述测试用例, 包含主车、道路基础设施、天气、交通参与者等动态及静态要素. 场景的定义在虚拟环境、现实环境或虚拟现实环境中同样适用.                                        |
| 情景集       | 一个或多个场景的集合.                                                                                                                                                                |
| 模型在环     | 将非常复杂的控制问题简化成可视化模型, 对模型进行仿真测试, 通过输入一系列的测试用例, 验证算法模型是否准确地实现了功能需求.                                                            |
| 软件在环     | 将编译后的代码通过一系列测试用例进行仿真测试, 验证代码实现的功能.                                                                                                                    |
| 硬件在环     | 将真的控制器连接实时仿真, 通过一系列测试用例, 验证代码实现的功能.                                                                                                                    |
| 车辆在环     | 在 ECU/EPP/整套系统 上, 通过一系列测试用例, 验证实现的功能是否与需求定义一致                                                                                                           |
| 状态机       | 通过定义信号接口触发条件, 实现 ADAS/AD 系统状态机状态切换, 进行人机交互.                                                                                                               |
| 交通流       | 除主车外的交通动态参与者.                                                                                                                                                            |
| 调度模式同步 | 每一帧按照模组配置中模块的顺序依次调度各个模块, 第一个模块执行完再执行第二个模块, 所有模块执行完再进入下一帧.                                                                        |
| 调度模式异步 | 每一帧按照模块配置中的调度周期同时调度起所有模块开始执行, 并且当前帧不会等待模块执行完成, 进入下一帧后根据当前仿真时间和每个模块 配置中的预期执行时间决定等待收取哪些模块的执行结果. |
| WorldSim     | 指运行有一定测试逻辑定义的人工构建虚拟场景仿真 测试类型.                                                                                                                             |
| LogSim       | 指测试数据为基于实采路测数据回灌的仿真测试类型.                                                                                                                                      |
