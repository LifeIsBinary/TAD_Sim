class ResetHelper

Public Functions

inline std::string map_file_path()

获取当前场景对应的地图文件路径。

返回:

当前场景对应的地图文件的绝对路径。

inline Vector3d map_local_origin()

获取当前场景对应的地图的建议原点。仿真系统中发出的障碍物及主车信息中的位置 一般采用经纬度形式，通过该原点可以方便的将这些经纬度坐标转换为局部坐标系以便于 计算。

返回:

当前场景对应地图的原点坐标(经度-纬度-海拔)。

inline Vector3d ego_destination()

获取当前场景主车的终点位置(经纬度)。

返回:

当前场景主车的终点坐标(经度-纬度-海拔)。

inline double ego_speed_limit()

获取当前场景限速信息。该限速区别于地图数据中不同路段的限速信息，通常是用户在 场景编辑器中针对主车在当前整个场景中设置的一个最大速度，用户算法需要同时考虑 地图元素和当前场景中的限速信息。

返回:

当前场景中针对主车的限速值(最大速度)，单位为米/秒(m/s)。

inline const std::string ego_start_location()

获取当前场景中主车的起始定位信息。如果当前模块订阅了系统内置的定位消息 (topic LOCATION)，则和在第0帧的Step方法中获取到的定位消息相同。返回的结果是Google的 protocol buffer 序列化后的字节数组， 详见 tx_sim::StepHelper::GetSubscribedMessage 。

返回:

经protocol buffer序列化后的主车初始定位消息。

inline std::vector<Vector3d> geo_fence()

获取本车的地理围栏信息，由返回的一系列经纬度坐标点构成的首尾相接的闭合多边形所表示。

返回:

构成主车地理围栏边界的经纬度坐标点的有序列表，长度至少为3，首尾相接。

inline std::string scenario_file_path()

获取当前场景对应的场景文件路径。

返回:

当前场景对应的场景文件的绝对路径。

inline std::vector<Vector3d> ego_path()

获取主车路径的控制点信息，每个控制点由经纬度以及高程组成。 第一个控制点为主车起点，同 ego_start_location 中的经纬度信息，最后一个控制点为主车的终点，同 ego_destination ，其余为主车路径中的途径点。

返回:

主车路径的控制点列表，列表中的元素为浮点类型的三元组，分别代表经度、纬度、高程。

inline void vehicle_measurements(std::vector<std::pair<int64_t, std::string>> &measurements)

获取当前场景中（多个）主车的ID和对应车辆尺寸信息，具体参考vehicle_geometry.proto中的定义。 返回时measurements数组中每个二元组第二项std::string是Google的 protocol buffer 序列化后的字节数组， 需要反序列化为vehicle_geometry.proto中定义的VehicleGeometoryList对象然后再获取各个尺寸信息， 详见 tx_sim::StepHelper::GetSubscribedMessage 。

inline const std::string scene_pb()

获取scene proto 返回的结果是Google的 protocol buffer 序列化后的字节数组

inline const std::string setting_pb()

setting proto 返回的结果是Google的 protocol buffer 序列化后的字节数组，反序列后返回的是相关配置，具体内容如下 versioin:版本号 scenario_path:场景文件路径 hadmap_path：地图路径 vehicle_dynamic_cfg_path：动力学参数配置路径 sensor_cfg_path：传感器配置路径 environment_cfg_path：环境参数配置路径 default_xosc_path：主车配置路径 scenario_labels：场景标签

inline const std::string group_name()

获取当前主车所在的组名

备注

对于多主车仿真每个主车算法会有一个对应的组,该组名会在创建场景的时候写到场景文件里面, 对于旧的场景文件没有该字段的默认返回Ego001。

返回:

主车的组名
